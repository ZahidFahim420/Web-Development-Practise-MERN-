‚úÖ 3. Render Tree Construction
The browser combines the DOM and CSSOM to form the Render Tree.

This tree contains only visible elements (excluding <head>, hidden elements, etc.).

Each node contains style and layout information.

‚úÖ 4. Layout (Reflow)
The browser calculates the size and position of each element.

This step considers:

Box model

Positioning (static, absolute, fixed, etc.)

Display types (block, inline, flex, etc.)

‚úÖ 5. Painting
The browser converts each render tree node into pixels.

Paints visual styles (colors, borders, shadows, etc.) to layers.

‚úÖ 6. Compositing
Layers (especially with effects like transforms, z-index, opacity) are combined into the final image.

GPU can be used to optimize performance.
